# RE:TOUR

## Project info

This is the RE:TOUR application, a Vite + React project.

## How to run locally

**Prerequisites**
- Node.js & npm installed

**Setup**
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development environment:
   ```bash
   npm run dev:all
   ```
   This will start the web server and any other required services with prefixed logs.

## Build

To build for production:
```bash
npm run build
```
